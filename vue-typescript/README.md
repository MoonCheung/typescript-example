# vue-typescript

> 不使用 Components-api 安装包，而是该项目基于 Vue3+Typescript，使用内置 Components-api 来实践代码学习

## Project setup

```
# Install dependency package
yarn

# Compiles and hot-reloads for development
yarn serve

# Compiles and minifies for production
yarn build

# Lints and fixes files
yarn lint
```
